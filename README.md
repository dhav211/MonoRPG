This was one of my first engineering feats I was proud of. A completely unplayable roguelike built with Monogame. I felt so proud not using an engine. You can see I had a whole entity system where you could add components, modeled after Unity. Reading through the code it's crazy inefficient but it was so much fun to build, and I learnt so much in the process.

There's absolutely no reason to run this, this is merely an artifact on my journey becoming a programmer.
